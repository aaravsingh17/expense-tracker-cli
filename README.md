# Expense Tracker CLI

A command-line expense tracker built in Python that helps you manage and analyse your spending.

## Features
- Add expenses with date, category, description and amount
- View all expenses in a formatted table
- Summarise spending by category with a visual bar chart
- Delete expenses by ID
- Export expenses by date range to a CSV file

## Categories
Food, Transport, Shopping, Bills, Other

## How to Use
```bash
python project.py add       # Add a new expense
python project.py view      # View all expenses
python project.py summary   # View spending summary
python project.py delete    # Delete an expense
python project.py export    # Export expenses to CSV
```

pip install tabulate
